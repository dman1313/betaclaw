# HEARTBEAT.md - Maintenance Checklist

_Run during heartbeat polls. Pick 1-2 items per heartbeat, rotate._

## Session Startup
1. Read `SOUL.md`, `USER.md`, today + yesterday memory
2. Read `MEMORY.md` (main session only)

## Periodic Maintenance (rotate through)
- [ ] **Memory consolidation** — Review recent daily files, distill into MEMORY.md
- [ ] **Failure review** — Check `memory/failures.md`, extract lessons
- [ ] **Skill audit** — Review `skills/` directory, update stale skills
- [ ] **Project check** — Review `memory/projects/`, flag stale items
- [ ] **Preference update** — Review recent chats, update `memory/preferences.md`
- [ ] **Git housekeeping** — Commit any uncommitted changes

## End-of-Session (when session was productive)
- [ ] Extract new skills from conversation
- [ ] Update project state files
- [ ] Log any new preferences discovered
- [ ] Document any failures or lessons
- [ ] Update MEMORY.md with significant events
